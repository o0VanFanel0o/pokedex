# Pokédex 🔥

Aplicación web que consume la API de Pokémon para mostrar los primeros 151 Pokémon con búsqueda dinámica.

## 🛠️ Tecnologías

- HTML5

- Mostrar 151 Pokémon
- Búsqueda en tiempo real
- Tarjetas dinámicas
- Consumo de API externa
